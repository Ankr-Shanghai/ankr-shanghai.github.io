    You can use the send function to transfer nft

    Switch To NFT Tab #

    Select NFT #

    Select the NFT you want to send and click

    Click Send #

    On the NFT details page, click the send button

    Enter Dstination Address #

    After entering the transfer destination address, click the send button.

    Select Payment Method #

    Select the payment method you need and click the submit button

    Native Payment

    Use native tokens to pay fees

    Erc20 Paymaster

    Use other tokens to pay fees

    Waiting For Submission Result #

    Please pay attention to whether the balance is sufficient before submitting

    If the submission is successful, the page will automatically return to the homepage, otherwise it will stay on this page.

    You can go to the Activity page to view the final send result.
