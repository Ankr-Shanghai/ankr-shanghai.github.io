    Switch To Activity Tab #

    View List #

    Find the data you want to view in the Activity list

    View Detail #

    Click on the data you want to view. In the pop-up window, you can see the transaction hash and user operation hash.

    You can click on the hash to open the corresponding browser to view the details of the transaction.

    Clicking on the transaction hash will jump to the blockchain browser of the corresponding network.

    Clicking on user operation hash will jump to our ERC-4337 UserOps Explorer
