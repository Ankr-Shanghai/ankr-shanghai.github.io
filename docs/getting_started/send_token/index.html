    You can use the send function to transfer token

    Select Token #

    Quick Send Native Token #

    Click the “send”

    Standard Send ERC-20 Token #

    Click on the token you want to send in the token list

    Click the “send”

    Enter Destination Address And Amount #

    After entering the transfer destination address and the amount to be transferred, click the send button.

    Address is the destination address to be send

    Asset is your token information, including token name and balance

    Amount is the number of tokens you want to send

    Select Payment Method #

    Select the payment method you need and click the submit button

    Native Payment

    Use native tokens to pay fees

    Erc20 Paymaster

    Use other tokens to pay fees

    Waiting For Submission Result #

    Please pay attention to whether the balance is sufficient before submitting

    If the submission is successful, the page will automatically return to the homepage, otherwise it will stay on this page.

    You can go to the Activity page to view the final send result.
